================================================================================
  PASO 3: CONFIGURAR API EN RENDER
================================================================================

1. Ve a Render Dashboard: https://dashboard.render.com
2. Ve a tu servicio de API (o créalo si no existe)

3. En "Settings" → "Build & Deploy":
   
   Root Directory: source
   (Solo la palabra "source", sin barras, sin puntos)
   
   Build Command: pip install -r requirements.txt
   
   Start Command: gunicorn api_server:app --config gunicorn_config.py

4. En "Environment":
   Agrega variable:
   API_SECRET_KEY = [genera una clave aleatoria]
   
   Para generar clave:
   - Usa: https://randomkeygen.com/
   - O ejecuta en Python: import secrets; print(secrets.token_hex(32))

5. Guarda los cambios
6. Render reiniciará automáticamente

================================================================================
  PASO 4: CONFIGURAR SERVICIO WEB PARA USAR LA API
================================================================================

1. En Render, ve a tu servicio WEB (no el de API)
2. En "Environment", agrega:
   API_URL = https://aspers-api.onrender.com
   (Reemplaza con la URL real de tu API)

3. Guarda los cambios
4. El servicio web se reiniciará
